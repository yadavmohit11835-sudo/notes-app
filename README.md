# Secure Notes App
A full stack notes application built using React, Node.js, Express, and MongoDB.

## Features
- User Authentication (JWT)
- Add Notes
- View Notes
- Delete Notes
- Secure API

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT

## How to Run
### Backend
cd notes-backend
node server.js

### Frontend
cd notes-frontend
npm run dev
